## What it will do

Install the AnonAge app from the Shopify App Store, pick which products or collections need an age
check, and the storefront gate appears automatically. No theme editing and no API keys to handle —
the app provisions and stores them for you.

<CardGroup cols={2}>
  <Card title="Gate by product" icon="tag">
    Apply the age requirement to specific products, collections, or the whole store.
  </Card>
  <Card title="Set the age" icon="calendar">
    Choose the minimum age per rule — 16, 18, 21, or a custom value.
  </Card>
  <Card title="Remembers customers" icon="user-check">
    A verified shopper is not asked again on their next visit.
  </Card>
  <Card title="Usage in one place" icon="chart-line">
    Verifications from your store roll up into the same Dashboard usage reports.
  </Card>
</CardGroup>
